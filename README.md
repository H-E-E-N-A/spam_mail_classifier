Spam Mail Classifier with Machine Learning

This project implements a machine learning model to classify emails as Spam or Ham (legitimate).

Files

spam_classifier.py: The Python backend script using Scikit-Learn (Multinomial Naive Bayes).

App.jsx: A professional React dashboard to interact with the classifier.

README.md: This documentation.

How to Run

Run the Python script: python spam_classifier.py

Run the UI: Use a React environment or the Preview tool.

Requirements

Python 3.x

Pandas

Scikit-Learn
